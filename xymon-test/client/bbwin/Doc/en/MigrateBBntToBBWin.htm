MigrateBBntToBBWin

Description

Tool to migrate configuration from bbnt client to the BBWin format.

Usage

cscript MigrateBBntToBBWin.vbs OutPutBBWin.cfg

This will generate 2 files :

  • OutPutBBwin.cfg : contains the XML BBWin.cfg file converted from bbnt registry
  • OutPutBBwin.cfg.reg : contains the registry key for the hostname setting
