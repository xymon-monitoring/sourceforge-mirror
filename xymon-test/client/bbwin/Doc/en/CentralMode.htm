BBWin Centralized Mode (Hobbit Mode) TESTING

Modes

BBWin can work with 2 modes :
  • the local mode : used by default. Status are sent to the Hobbit/BigBrother server
  • the centralized mode : to be used with the Hobbit server. a report is send to the hobbit server via the clientdata channel
To enable the centralized mode, check the mode option in the BBWin.cfg file.

Notes

  • Some agents which are not compatible with the centralized mode will be launch with the local mode. Examples (externals)
  • The centralized mode use the bbwin tmp directory to save the report file and the client-local.cfg file from the hobbit server

Example :

<?xml version="1.0" encoding="utf-8" ?>
<configuration>
<bbwin>
	...
	<setting name="mode" value="central" />
	<setting name="configclass" value="win32" />
	...
</bbwin>
