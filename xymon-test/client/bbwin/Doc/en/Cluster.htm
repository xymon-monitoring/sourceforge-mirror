Cluster

Description

External executable to monitor MSCS clusters. It will report alerts you when :

  • the active node has changed
  • one or may group ressources are not online on the active node

To be sure to get the alert of a node change, it will still report the alarm until you delete a flag file that was created when the cluster has moved to the other node.

MSCS Cluster external report only yellow color.

Usage

Add the executable in a load directive for externals agent. It will generate a report as others externals scripts. There are no setting available for this external.

Examples

  • Here an example of report when the active node has changed : 

    
--- Windows Cluster State ---

    

    Group NameCurrent NodeStateStatus
    


    Cluster GroupMYSERVERBDD02 (recently changed)online
    

    SQLMYSERVERBDD02 (recently changed)online
    

    


Important Notes :

WARNING !!! active node for group "Cluster Group" has changed. It was MYSERVERBDD01 and now it is MYSERVERBDD02.
To suppress this warning, please connect to the server and execute the command : erase "C:\WINNT\TEMP\NodeChanged_Cluster Group"

WARNING !!! active node for group "SQL" has changed. It was MYSERVERBDD01 and now it is MYSERVERBDD02.
To suppress this warning, please connect to the server and execute the command : erase "C:\WINNT\TEMP\NodeChanged_SQL"

Notes

  • Need Windows 2000 Advanced Server or Windows 2003
  • This external is not installed by default
