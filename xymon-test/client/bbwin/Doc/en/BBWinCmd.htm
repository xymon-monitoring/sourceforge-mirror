BBWinCmd

Description

BBWinCmd is a diagnostic/testing command line tool for the Big Brother / Hobbit protocol. It allow you to send messages to the hobbit server as the bb command under Unix. BBWinCmd is installed by default in the BBWin binary directory.

Usage

bbwincmd <options...>

Sending a status :

bbwincmd.exe <bbdisplay>[:<port>] status <hostname> <testname> <color> <text> [<lifetime>]

Sending a data :

bbwincmd.exe <bbdisplay>[:<port>] data <hostname> <dataname> <text>

Sending a notify :

bbwincmd.exe <bbdisplay>[:<port>] notify <hostname> <testname> <text>

Sending a disable:

bbwincmd.exe <bbdisplay>[:<port>] disable <hostname> <test> <duration> <text>

Sending an enable:

bbwincmd.exe <bbdisplay>[:<port>] enable <hostname> <test>

Sending a query and get the result:

bbwincmd.exe <bbdisplay>[:<port>] query <hostname> <test>

Sending a config and get the file content:

bbwincmd.exe <bbdisplay>[:<port>] config <filename>

Sending a hobbit message manually written

bbwincmd.exe <bbdisplay>[:<port>] message <message>

Sending a drop

bbwincmd.exe <bbdisplay>[:<port>] drop <hostname> [<testname>]

Sending a hostname rename

bbwincmd.exe <bbdisplay>[:<port>] rename <hostname> <newhostname>

Sending a test rename

bbwincmd.exe <bbdisplay>[:<port>] rename <hostname> <oldtestname> <newtestname>

Notes

  • If no port is specified after bbdisplay, it will use hobbit tcp port 1984
  • You can set your default hobbit server by setting the environment variable BBDISPLAY. To use your environment variable, replace the <bbdisplay>[:<port>] argument by the character $
