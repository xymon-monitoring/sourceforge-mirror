Sample

Description

Sample agent is an example for people who would like to develop their own agents. It may be used as an example or as a code base. It actually send a simple green status report with a short message. Testname and message can be configured via the BBWin.cfg

Configuration

  • setting directive syntax : 

    <setting name="" value="" />

    name value
    message You can set the message that will be reported. Default is "this is a simple test"
    testname You can set the testname that will be sent to the monitoring server (column name in the bbdisplay) Default is "sample"

Examples

  • Sample will report a green status with the name "hello" and it will contain the text "Hellow World" 

    <sample>
	<setting name="message" value="Hello World" />
	<setting name="testname" value="hello" />
</sample>

Notes

  • This agent is not installed by default and is part of the BBWin Agent SDK
